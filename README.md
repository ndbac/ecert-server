# Introduction
ThisisBac backend services are responsible for all actions related to ThisisBac Blog by providing RESTful APIs and also custom scripts.

# Swagger Documentation Link
https://thisisbac.herokuapp.com/docs/api/

# Project Setup Instructions

## Installation

```bash
$ yarn install -force
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
